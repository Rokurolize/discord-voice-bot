# Discord Voice Bot

A Discord voice channel text-to-speech bot with Zundamon voice.

## Usage

```bash
uv run discord-voice-bot
```

## Development

```bash
# Run tests
uv run pytest

# Run linting
uv run poe lint

# Format code
uv run poe format
```

## Requirements

- Python 3.12
- Discord.py
- Various other dependencies (see pyproject.toml)
